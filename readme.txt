<h4>Features</h4>
- **Widget** 
- **Shortcode** 

== Installation ==

= Install = 

1. Upload **`github-buttons.php`** and other files compressed in the zip folder to the **`/wp-content/plugins/`** directory.,
2. Activate the plugin through the 'Plugins' menu in WordPress.
